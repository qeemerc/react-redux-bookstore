# react-redux-bookstore
Bookstore app with React, Redux, Router - [Live Demo](https://react-redux-bookstore.netlify.com)

**Features:**

* MockApi
* Adaptive (>= 500px width)
* Infinity load
* Filter by caregory and name

**Libs:**

* React
* Redux
* React-Router
* Redux-Thunk
* Redux-Logger
* PropTypes
* Axios
* Reselect
* Semantic UI
